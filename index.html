Projects

  • The Legend of Zelda (Unity Remaster)

Welcome to a remaster of the NES version of The Legend of Zelda. It was developed using the Unity engine, Visual Studio for scripts written in C#, and Photoshop for sprite creation/edits. I was responsible for most of the enemy creation along with small portions of the game mechanics and level design. My partner did all of the creation on the custom level.

  • Controls:
  • arrow keys/WASD: movement
  • x: primary weapon
  • z: secondary weapon
  • spacebar: cycle secondary weapons
  • 1: god mode
  • 4: custom level
  • Shape-Drop

Welcome to Shape-Drop. Shape-Drop is a game where you are an arrow and you have to make it through holes in platforms that are coming at you. This is a game that I solo developed over the course of two weeks.

  • Controls:
  • Keyboard
  • Arrow keys/WASD for movement
  • 1: up arrow
  • 2: right arrow
  • 3: down arrow
  • 4: left arrow
  • Controller:
  • Left joystick for movement
  • X: up arrow
  • A: right arrow
  • B: down arrow
  • Y: left arrow
